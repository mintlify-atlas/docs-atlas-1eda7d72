    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get MTGBuilder up and running in three simple steps.
  </p>

  <Steps>
    <Step title="Install MTGBuilder">
      Install MTGBuilder via Cabal from Hackage. You'll need the Haskell Platform installed on your system.

      ```bash
      cabal install MTGBuilder
      ```

      This installs the `mtg-builder` command-line tool globally.
    </Step>
    
    <Step title="Prepare your input decks">
      Create deck list files in the standard MTG format. Each file should contain one deck list with card names and quantities.

      ```text deck1.txt
      4 Lightning Bolt
      4 Snapcaster Mage
      3 Cryptic Command
      2 Jace, the Mind Sculptor
      ```

      You can use any number of input deck files to analyze different archetypes or variations.
    </Step>
    
    <Step title="Generate your deck">
      Run MTGBuilder with your input files to generate an optimized deck based on card popularity rankings.

      ```bash
      mtg-builder deck1.txt deck2.txt deck3.txt -o output.txt -r 3
      ```

      <Accordion title="Example Output">
        The tool will output a 60-card mainboard and 15-card sideboard based on the ranking analysis:
        
        ```text
        4 Lightning Bolt
        4 Snapcaster Mage
        3 Cryptic Command
        2 Jace, the Mind Sculptor
        ...
        SB: 2 Wear // Tear
        SB: 3 Dispel
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
